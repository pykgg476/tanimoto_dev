たにもと建設は、海と人々をつなぐ建設のプロフェッショナルです。

たにもと建設は創業50年の歴史をもつ、横浜での港湾土木を中心とした建設会社です。

船舶、潜水、溶接など港湾土木に必要な様々な専門技術を持つ会社を取りまとめ、安全・工程・品質管理等の施工管理業務を行う会社です。

「あなたと海をつなぐ。」をスローガンとし、日々、人々の暮らしの基盤づくりに貢献しております。
